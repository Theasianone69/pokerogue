    Function getEnumKeys

    • getEnumKeys<E extends EnumOrObject>(enumType: TSNumericEnum<E>): (keyof E)[]

      Return the string keys of an Enum object, excluding reverse-mapped numbers.

      Type Parameters

      • E extends EnumOrObject

      Parameters

      • enumType: TSNumericEnum<E>

        The numeric enum to retrieve keys for

      Returns (keyof E)[]

      An ordered array of all of enumType's string keys

      Example

      enum fruit {
        apple = 1,
        banana = 2,
        cherry = 3,
        orange = 12,
      };

      console.log(getEnumKeys<typeof fruit>(fruit)); // output: ["apple", "banana", "cherry", "orange"]

      Remarks

      To retrieve the keys of a NormalEnum, use Object.keys instead.
